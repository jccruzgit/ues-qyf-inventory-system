# UES QYF Inventory System

Sistema de Inventario para la Facultad de Química y Farmacia de la Universidad de El Salvador.

## Tecnologías previstas
- Java
- Spring Boot
- PostgreSQL
- Flyway
- React
- Tailwind CSS

## Estructura del proyecto
- `backend/`: API y lógica de negocio
- `frontend/`: interfaz web del sistema
- `docs/`: documentación del proyecto
- `database/`: diagramas, migraciones y scripts

## Estado actual
Proyecto en fase de diseño y preparación de backend.
